# Диагностические утилиты

Эта директория содержит скрипты для диагностики и отладки системы.

## Доступные утилиты

### `diagnose_metrics.sh`

Скрипт для диагностики метрик на сервере.

**Использование:**

```bash
# Скопируйте на сервер
scp scripts/diagnose_metrics.sh user@server:/opt/newsagent/

# На сервере
cd /opt/newsagent
chmod +x diagnose_metrics.sh
./diagnose_metrics.sh
```

**Что проверяет:**
- ✅ Статус контейнера
- ✅ Логи контейнера
- ✅ Доступность порта 8000
- ✅ Метрики изнутри контейнера
- ✅ Метрики с хоста
- ✅ Переменные окружения
- ✅ Процессы внутри контейнера
- ✅ Сетевые подключения
- ✅ Конфигурацию docker-compose

## Добавление новых утилит

При добавлении новых диагностических скриптов:

1. Разместите их в этой директории
2. Сделайте исполняемыми: `chmod +x script.sh`
3. Добавьте описание в этот README
4. Обновите документацию в `docs/` если нужно
